This is an online book review website where users need to register for accounts to be able to rate books.  

The book has one publisher and is authored by one or many people where each book has unique book ID, title and published year. 

Authors are identified by author ID and have a name. 

Publishers are identified by publisher ID and have a name. 

Genres have a unique name with description. A book can have many genres. 

Users can review on many books and rate on each book one time. Each user has a unique ID, name, DOB and unique email address. There are two types of users, professional and amateur.
